# 摘要

非物质文化遗产数字化传播面临交互方式单一、推荐结果缺乏解释、冷启动明显以及用户兴趣难以持续沉淀等问题。针对这些问题，本文设计并实现了一个融合 AI 数字人导览与对话推荐机制的非遗文化传播微信小程序平台。系统以 AI 数字人“黑塔”为核心交互入口，引入对话推荐系统（CRS）思想，通过 ASK-REC 状态机、三维置信度模型、可解释推荐与行为回流机制，构建了“AI 导览—推荐执行—行为反馈—画像更新”的闭环。

本文的主要工作包括：（1）设计并实现了面向非遗场景的轻量个性化推荐引擎，将显式偏好、浏览记录、AI提问、活动报名、社区互动与当前对话上下文六维信号共同纳入画像构建，并对内容、活动、讨论三类对象分别进行排序；（2）设计了基于三维置信度的 CRS 对话推荐机制，通过 ASK-REC 状态机实现冷启动提问、混合推荐、精准推荐、意图驱动推荐和恢复提问等策略切换，使系统能够根据用户理解程度动态决定“先问还是先推”；（3）实现了推荐可解释性机制，系统不仅输出自然语言推荐理由，还通过四层解释结构（用户可读理由、系统依据、策略上下文、知识图谱推理）返回匹配分、新鲜度、惩罚项、决策模式、图谱路径等结构化解释字段；（4）构建了知识图谱增强模块和行为闭环诊断能力，使系统能够在 AI 问答中识别实体、扩展候选方向、生成路径说明，并通过日志统计观察 ASK 回答率与跳过率、模式分布、冷启动收敛情况、ASK模板级运营统计以及行为对置信度和隐式偏好的影响。

系统前端采用微信小程序原生开发，后端基于 Python、FastAPI 与 SQLAlchemy 实现，数据库采用 SQLite 进行轻量部署。系统测试与实验结果表明：平台主要功能运行稳定；ASK-REC 策略能够显著改善冷启动推荐效果；知识图谱增强能够提升推荐解释性与多样性；行为回流能够进一步推动用户画像更新与 CRS 模式升级。该系统验证了在毕业设计规模下构建可运行、可解释、可观测的非遗对话推荐平台具有可行性。

**关键词：** 非物质文化遗产；对话推荐系统；AI数字人；知识图谱；可解释推荐；微信小程序

# Abstract

The digital dissemination of intangible cultural heritage (ICH) faces several challenges, including limited interaction modes, weak recommendation explainability, prominent cold-start problems, and difficulty in continuously accumulating user preferences. To address these issues, this thesis designs and implements a WeChat Mini Program platform for ICH communication that integrates an AI digital human guide with a conversational recommendation mechanism. Centered on the AI guide “Heita,” the system introduces the concept of Conversational Recommendation Systems (CRS) and builds a closed loop of “AI guidance → recommendation execution → behavioral feedback → profile update” through an ASK-REC state machine, a three-dimensional confidence model, explainable recommendations, and behavior feedback logging.

The main contributions of this work are as follows: (1) A lightweight personalized recommendation engine for the ICH scenario is designed and implemented, integrating six-dimensional signals — explicit preferences, browsing records, AI questions, activity registrations, community interactions, and current dialogue context — into user profile construction, and separately ranking content, activities, and discussion topics; (2) A CRS mechanism based on a three-dimensional confidence model is proposed, where the ASK-REC state machine supports strategy switching among cold-start questioning, mixed recommendation, precision recommendation, intent-driven recommendation, and recovery questioning, enabling the system to dynamically decide whether to ask first or recommend first; (3) An explainable recommendation mechanism is implemented, where the system returns not only natural-language recommendation reasons but also a four-layer explanation structure (user-readable reason, system basis, strategy context, knowledge-graph reasoning) with structured fields such as match score, novelty score, penalty term, decision mode, and knowledge-graph path; (4) A knowledge-graph enhancement module and a behavior-loop diagnosis mechanism are constructed, allowing the system to recognize entities in AI conversations, expand candidate directions, generate reasoning paths, and observe ASK answer rate, mode distribution, cold-start convergence, and the influence of user behavior on confidence and implicit preference scores through logs and statistics.

The frontend is implemented with native WeChat Mini Program development, while the backend is built with Python, FastAPI, and SQLAlchemy, using SQLite for lightweight deployment. Testing and experimental results show that the platform runs stably, the ASK-REC strategy significantly improves cold-start recommendation performance, knowledge graph enhancement improves recommendation explainability and diversity, and behavior feedback further promotes user profile updates and CRS mode upgrades. The system demonstrates the feasibility of building a runnable, explainable, and observable conversational recommendation platform for ICH within the scale of an undergraduate graduation project.

**Keywords:** Intangible Cultural Heritage; Conversational Recommendation System; AI Digital Human; Knowledge Graph; Explainable Recommendation; WeChat Mini Program
