哲学一致性

7
Strong · 统一方向明确

暗色水墨渐变背景(175deg, #1a0f07→#9c4e15)贯穿全页，配合回纹边框、旋转光环、印章徽标三个装饰体系，共同营造"推开非遗博物馆大门"的氛围感。三个特性卡片(壹/贰/叁)使用统一的左色条+暗底布局，字体选择一致收敛到衬线标题+无衬线正文。唯一的偏离：登录按钮渐变从 135deg 红渐变变成纯红+金光晕，与背景 175deg 的斜向不一致(差了 40deg)，这是一个微小的方向漂移。

✓ Keep暗色水墨背景+回纹+光环三件套装饰体系
⚡ Quick-win统一背景和按钮渐变角度到 135deg

视觉层级

5
Functional · 核心流程可用但信息排序混乱

第一眼看到的是旋转光环+数字人(纯视觉装饰，无信息)，其次是品牌标题(42px)，再到三个特性卡片，最后才到表单输入框。用户要做的事(登录)在视觉层级上排第四位。特性卡片虽然精美但并非用户首要目标——它抢占了"输入用户名"之前的位置。在手机屏幕上(390×844)，表单输入框需要滚动才能看到，对于注册/登录这样的核心CT A来说层级过低。标题42px与输入框15px的对比度也过大(2.8:1)，造成"头重脚轻"。

⚠ Fix将登录表单上移到品牌标题正下方，特性卡片移到表单之后
⚡ Quick-win减小标题字号至32px、增大输入框字号至16px，缩小头脚差距

细节执行

6
Functional · 大部分细节到位，2-3处粗糙

输入框的 rounded-full + 半透明背景 + 金色微边框组合精致(line 57)。回纹边框使用 repeating-linear-gradient 模拟回纹效果准确。但三个特性卡片的背景 bg-black/20 导致了"悬浮无根"的漂浮感——卡片没有阴影，像是浮在背景上而非嵌入页面。游客体验按钮的 bg-white/[0.07] 与输入框背景相同，但语义不同(一个可点击一个可输入)，视觉上不应相同。底部"注册/登录"切换文字颜色 text-amber-200/60 在深色背景上对比度不足(约 3.2:1，低于 WCAG AA 标准)。

✓ Keep输入框样式——半透明+金色边框，精致且与背景统一
⚠ Fix特性卡片添加微阴影(box-shadow)，增强嵌入感而非漂浮感

功能性

7
Strong · 核心流程完善

游客一键登录(调用 /auth/guest)、注册/登录切换、测试账号一键填入、表单校验、加载态禁用、错误提示——所有核心认证流程都到位。输入框使用原生 required 属性做基础校验。装饰元素全部设置 pointer-events-none(line 64,66,70,72)避免了遮挡点击的常见陷阱。加载中按钮显示"处理中..."并提供视觉反馈。缺少：密码可见性切换(👁)、回车键提交的明确提示、网络断开时的优雅降级提示。

✓ Keeppointer-events-none 装饰元素 + 测试账号一键填入
⚡ Quick-win添加密码可见性切换按钮(15分钟)

创新性

7
Strong · 有多个值得借鉴的设计手法

三个创新点值得注意：(1) CSS纯代码绘制数字人(黑塔)直接嵌入登录页，而非通用的插图占位——这是独特的品牌资产；(2) 旋转光环+四角星芒装饰——用 CSS animation 而非静态图实现动态装饰，且性能开销极小；(3) 回纹边框——用 repeating-linear-gradient 模拟中国传统回纹，是文化元素的数字化转译。这三个手法都服务于"非遗"主题，属于 earned innovation。登录按钮的红→金渐变过度设计(之前四级过渡)，现已简化为纯红+金光晕，更好。

✓ KeepCSS数字人+旋转光环+回纹边框：三大品牌创新资产
⚡ Quick-win给光环添加两个小型轨道粒子(◆✦)，增强动态感

行动清单

KEEP 保持 —— 这些不要动

  • 暗色水墨背景 (175deg, #1a0f07→#9c4e15) — 氛围基调正确
  • 回纹边框 + 旋转光环 + CSS数字人 — 三大装饰资产
  • 输入框样式 (半透明+金色微边框+圆形) — 精致且可用
  • 所有装饰元素 pointer-events-none — 避免遮挡点击
  • 游客体验 + 测试账号填入 — 便捷的测试入口

FIX 修复 —— P0/P1 视觉代价高

  • P0 将登录表单上移到品牌标题下方（当前表单在特性卡片之后，用户需滚动）
  • P1 特性卡片添加微阴影 (box-shadow: 0 4px 16px rgba(0,0,0,0.15))
  • P1 游客按钮改用独立视觉样式（不要和输入框一样）
  • P1 底部切换链接文字对比度提升到至少 4.5:1

QUICK WINS 快速见效 —— 每个 5-15 分钟

  • 将特性卡片从 bg-black/20 升级为 bg-white/[0.06] + backdrop-blur
  • 给旋转光环添加 2 个小型轨道粒子 (◆✦)，增强动态精致感
